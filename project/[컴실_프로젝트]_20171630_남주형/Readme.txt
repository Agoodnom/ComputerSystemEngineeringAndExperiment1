프로젝트 환경
visual studio 2017 기반의 OpenFramwork(v0.11.0)

실행 방법
1. projectGenerator.exe를 사용하여 project를 생성한다.
2. 생성된 project 폴더에 있는 src폴더를 클릭한다.
3. 생성된 파일3개를 지워주고 제출한 프로젝트 code 5개
(main.cpp, ofApp.cpp, ofApp.h, ofxWinMenu.cpp, ofxWinMenu.h)를 src폴더에 넣어준다.
4. 생성된 project폴더에 있는 bin폴더를 클릭한다. 그리고 bin폴더에 있는 data폴더를 클릭한다.
5. data 폴더에 제출한 프로젝트 data 7개(.maz파일 4개, .png파일 3개)를 넣어준다.
6. 솔루션 탐색기에서 src에 추가->기존항목 추가를 선택하여 src파일 안에 있는 5개 파일을 추가한다.
7. 로컬 Windows 디버거를 실행하고 기다리면 창이 뜬다.

미로 파일 추가 방법
1. 창 왼쪽 상단에 있는 File 메뉴를 클릭한후 Open을 클릭한다.
2. Select .maz file 창이 뜨면 생성한 project 폴더->bin폴더->data폴더 속에 있는 
.maz 파일 중 하나를 선택한다.(level은 난이도 즉, 미로의 크기에 따라 정해졌다.)
3. 파일을 열면 미로가 그려진다.

플레이 옵션 선택방법
1. 창 왼쪽 상단 View 메뉴를 클릭하면 Play 1p와 Play 2p 옵션이 있다.
2. Play 1p는 1인용 play, Play 2p는 2인용 게임이다. 원하는 play option를 선택하면 play 가능하다.

플레이 방법 & 조작법
1. 1인용 play에선 키보드 방향키로 상하좌우로 이동이 가능하다.
2. 2인용 play에선 키보드 방향키는 1p, 'w'(상),'s'(하),'a'(좌),'d'(우) 는 2p가 사용하는 key이다.
3. 1p는 빨간색 볼, 2p는 파란색 볼, 목적지는 노란색 볼로 나타나있다.
4. 각 플레이어는 자신의 볼을 움직여 노란색 볼(목적지)이 있는 곳까지 먼저 이동하면 승리한다.
5. 게임이 종료되면 플레이 옵션을 다시 선택하거나 미로파일을 
다른 파일로 교체하여 다시 플레이가 가능하다.

추가 Tip
1. 창 크기를 늘리거나 줄여 미로의 크기를 조절할 수 있다.
2. 왼쪽 상단에 View 메뉴에 있는 Full screen을 선택하면 전체화면으로 play할 수 있다.
3. Full screen 상태에서 ESC를 누르면 전체화면이 종료된다.
4. 전체화면이 아닌상태에서 ESC를 누르면 프로그램이 종료된다.
